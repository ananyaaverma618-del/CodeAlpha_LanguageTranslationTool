CodeAlpha Language Translation Tool

Project Description

This project is an AI-based Language Translation Tool developed using Python and the Deep Translator library.

Features

- Translate text between multiple languages
- Automatic language detection
- Fast and accurate translations
- User-friendly interface

Technologies Used

- Python
- Deep Translator
- Google Colab

How to Run

1. Install the required library:
   pip install deep-translator

2. Run the Python script.

3. Enter the text and select the target language.
